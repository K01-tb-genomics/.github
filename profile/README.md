# K01 — Linking within- and between-host dynamics in tracking recent *Mycobacterium tuberculosis* transmission

Research outputs from NIH/NIAID K01 K01AI153544 (PI: Marie Nancy Séraphin),
investigating within-host genetic diversity of *M. tuberculosis* using
whole-genome sequencing of sputum cultures from Ghana and Florida.
